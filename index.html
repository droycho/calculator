BMI Calculator

This button will create a series of pop-up windows that will allow the user to calculate their Body Mass Index (BMI) by entering their height (in) and wieght(pounds). JavaScript will run a promt when the user clicks the button above that will ask the user for their weight and multiply it by 0.45. Then it will ask the user to enter their height and multiply by 0.025. The last window will display the final result when the resulting weight value is divided by the resulting height value squared (weight/height^2). This is the the users BMI.
